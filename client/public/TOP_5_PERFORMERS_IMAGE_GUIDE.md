# Top 5 Performers Image Guide

## Image Placement
Place your Top 5 Performers image in the public folder with the following name:
```
/client/public/top-5-performers.jpg
```

## Image Specifications

### Recommended Dimensions
- **Width:** 1200px - 1600px
- **Height:** 600px - 900px  
- **Aspect Ratio:** 16:9 or 4:3 (landscape orientation)
- **Format:** JPG or PNG
- **File Size:** Keep under 500KB for optimal loading

### Image Content
Your image should include:
- All 5 top performers with their details (rank badges, names, course counts, points, progress bars)
- The same visual style as shown in your current design (rank badges in gold/silver colors, profile photos, progress bars)
- Clean, professional layout matching the existing card style
- High resolution for clarity on all devices

### Alternative Formats
If you prefer a different image format, you can use:
- `top-5-performers.png` (for transparent backgrounds)
- `top-5-performers.webp` (for better compression)

Just make sure to update the image source in the Leaderboard.jsx file accordingly.

## Current Image Path in Code
The image is referenced in:
```
/client/src/components/Leaderboard.jsx
Line: src="/top-5-performers.jpg"
```
